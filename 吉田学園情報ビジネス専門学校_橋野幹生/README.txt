□01_GODZZILA	---------------------------------------------------
作品名	: VS.GODZZILA
ジャンル: 3Dアクション
制作期間: 2021年12月中旬～2022年2月中旬（約3か月）
開発環境: C言語、DirectX9
ツール	: VisualStudio2015、Photoshop、メタセコイア

<一言コメント>
怪獣のモデルを制作したことがきっかけで作った作品です。
街を壊していくゲームなので、建物の残りHPをゲージで表現したり、爆発のアニメーションなどを実装し、
破壊していく様子を表現しました。
マウスでの操作や、敵の追尾機能などは自分で調べて実装しました。
